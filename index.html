SpringMVC Twitter Bootstrap Showcase

Overview

This fork replaces Spring MVC Showcase's JQuery-UI with Bootstrap, tomcat with embedded jetty, tiles with Sitemesh, and other cleanup tasks (commmon js file).

For a complex use of SpringMVC/JQuery/Bootstrap/Ajax, see Spring MVC/REST Best Practices

Major Versions used

Spring 3.2
Twitter Bootstrap 2.3

Live Demo

http://springmvc-twbootstrap-showcase.cloudfoundry.com

Run locally with Maven

  1. Build > $ mvn clean install
  2. Run > $ mvn jetty:run
  3. Access the deployed web application at: http://localhost:8080/spring-mvc-showcase/
